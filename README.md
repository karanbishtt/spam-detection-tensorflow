# Spam Detection using TensorFlow

This project implements a **Spam Detection system** that classifies text messages as **Spam** or **Not Spam** using **TensorFlow** and basic **Natural Language Processing (NLP)** techniques.

The project is developed as a learning-focused implementation to understand how text data can be preprocessed and used to train a neural network for binary text classification.
